Algorítimos e Lógica de Programação

A disciplina tem como objetivo ensinar aos alunos o raciocínio lógico, ou seja, conhecer a sequência de passos correta que possibilite a realização de determinadas tarefas e, no curso, aplicar este raciocínio para cada fim através de diferentes linguagens de programação linguagens de programação. Chamamos essa sequência de passos lógicos de Algoritmos.


O que são linguagens de programação?

São codificações capazes de transmitir ao computador ordens, expressões ou sequência de passos a serem realizados. Elas contêm sintaxes que o programador entenda, variando de acordo com cada linguagem (C, C++, C#, JAVA, etc.).


Praticamente tudo no computador tem um algoritmo para poder “funcionar”, ou seja, é necessário haver alguma programação sequencialmente correta de acordo com a linguagem utilizada, a qual é compilada e “traduzida” para a linguagem que a máquina é capaz de processar: 0 e 1 (sim ou não).
